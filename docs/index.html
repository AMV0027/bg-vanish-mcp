How It Works

Three simple steps — from image to transparent PNG, all on your own machine.

1

Send an Image

Your AI assistant sends the image — either a file path on your computer or a base64 string.

2

AI Removes Background

The U2NET model processes the image locally — no data ever leaves your machine.

3

Get Transparent PNG

The result is saved as a transparent PNG — on your filesystem or returned as base64.

Installation

Get started in under a minute. Python 3.10+ required.

Recommended Quick Install
terminal 
pip install bg-vanish-mcp

Then run bg-vanish-mcp to start the server.

Windows + DirectML

Works with any GPU (AMD, Intel, NVIDIA)

pip install bg-vanish-mcp[dml]
set USE_GPU=true
Linux / macOS + CUDA

NVIDIA GPU required

pip install bg-vanish-mcp[gpu]
export USE_GPU=true
Manual setup from source 
git clone https://github.com/your-username/bg-vanish-mcp.git
cd bg-vanish-mcp
python -m venv .venv
source .venv/bin/activate    # Linux / macOS
# .venv\Scripts\activate     # Windows
pip install -r requirements.txt
Verify it works: python test_server.py

(Requires cloning the repo first)

Works With Every MCP Client

One install, connect to any MCP-compatible AI assistant.

CD

Claude Desktop

Add to your claude_desktop_config.json
macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json
Linux: ~/.config/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "bg-vanish-mcp": {
      "command": "bg-vanish-mcp"
    }
  }
}

  }
}

Tools Reference

Two simple tools that your AI assistant can call.

remove_background

Process an image file on your computer and save the result.

input_path Path to image (.jpg, .jpeg, .png)
output_path Where to save (default: <input>_no_bg.png)
return_base64 Set to true to get result as base64
remove_background_base64

Process a base64-encoded image directly and get back a transparent PNG.

image_base64 Base64 image data (with or without data:image/...;base64, prefix)
Returns

Base64-encoded transparent PNG (data:image/png;base64,...)

GPU Acceleration

Speed up background removal with your GPU.

By default, processing runs on CPU. For faster results, install the GPU runtime for your system. The server automatically falls back to CPU if the GPU isn't available.

Windows (any GPU)

pip install onnxruntime-directml
set USE_GPU=true

Linux / macOS (NVIDIA)

pip install onnxruntime-gpu
export USE_GPU=true
